## Spesifikasi Logo

- **Format**: PNG (disarankan) atau SVG
- **Ukuran**: Minimal 200x200px (akan di-resize otomatis)
- **Background**: Transparan (untuk PNG)
- **Rasio**: 1:1 (square) atau sesuai logo asli
- **Kualitas**: High resolution untuk hasil terbaik

## Cara Menambahkan Logo

1. Siapkan logo bank dalam format PNG atau SVG
2. Rename file sesuai dengan nama di tabel di atas
3. Pastikan background transparan (untuk PNG)
4. Simpan file di folder: `dashboard/public/banks/`
5. Restart aplikasi (jika perlu)

## Catatan Penting

- Jika logo tidak tersedia, sistem akan otomatis menggunakan placeholder dengan nama bank + warna brand
- Logo akan otomatis di-resize untuk tampilan kartu (ukuran optimal: 200x200px)
- Format PNG dengan background transparan direkomendasikan untuk hasil terbaik
- Pastikan nama file sesuai persis dengan yang terdaftar di database

## Sumber Logo

Anda bisa mendapatkan logo bank dari:
- Website resmi bank
- Wikipedia
- Brand asset bank (jika tersedia)
- Atau buat sendiri sesuai brand guidelines bank
