# Logo Bank

Folder ini berisi logo untuk semua bank yang didukung.

## Format File
- Format: PNG atau SVG (disarankan PNG dengan background transparan)
- Ukuran: Minimal 200x200px (akan di-resize otomatis)
- Nama file: Sesuai dengan `logo_filename` di master data bank

## Cara Menambahkan Logo

1. Download atau siapkan logo bank dalam format PNG atau SVG
2. Rename file sesuai dengan nama di atas (contoh: `bca.png`)
3. Pastikan background transparan (untuk PNG)
4. Ukuran minimal 200x200px untuk kualitas baik
5. Simpan file di folder `dashboard/public/banks/`

## Catatan

- Jika logo tidak tersedia, sistem akan menggunakan placeholder dengan nama bank + warna brand
- Logo akan otomatis di-resize untuk tampilan kartu
- Format PNG dengan background transparan direkomendasikan untuk hasil terbaik
